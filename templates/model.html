The Model


Intro

A combined set of labeled fake news and real news was used to train a naive bayes model to predict the probability that the article is fake based on its text content. The model was predicting with an 85% accuracy among the ~1,000 articles in our testing set with training set of ~4,000 articles.

Method

1. Real and fake articles are randomly selected for training and test set

2. Training set of articles are then fed into a count vectorizer to tokenize the text content

3. TF-IDF is run on the document-words matrix

4. The top 7000 words are collected into a vocabulary and used for future count vectorizers

5. Data is then used fitted to a naive bayes model
